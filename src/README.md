----
This app is helping he user to remember new words when learning a new language.

1. Repeat with 1-3-7-14-30 days routine.
2. Mark new words with personal memory, like an emoji.
3. Practice words with controller supported.


---
Run app

In powershell,
npm run start
- If npm run start gives "running scripts is disabled on this system", try command
Get-ExecutionPolicy
If it gives:
Restricted
Then try 
Set-ExecutionPolicy RemoteSigned
